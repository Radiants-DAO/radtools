## What's Included

### Components
The complete RadTools UI component library with:
- Buttons, Cards, Inputs, Selects
- Dialogs, Sheets, Popovers
- Tabs, Accordions, Tooltips
- Feedback components (Toast, Alert, Badge)

### DevTools
Visual development panel for:
- Design token management
- Typography editing
- Component discovery
- Asset management
- Mock state presets

### Claude Code Integration
- **Skills**: Quick reference for component APIs
- **Agents**: Code quality reviewer, documentation generator

## Usage

Import components from `@/components/ui`:

```tsx
import { Button, Card, Dialog } from '@/components/ui';
```

Toggle DevTools with `Shift+Cmd+K` (Mac) or `Shift+Ctrl+K` (Windows).
